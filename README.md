# HomeLab

Proyecto personal de infraestructura para aprender, documentar y construir un servidor casero basado en Linux, Docker, NAS y virtualización.

## Objetivos

- Aprender Docker y Docker Compose.
- Construir un entorno HomeLab ordenado y documentado.
- Versionar la infraestructura con Git y GitHub.
- Implementar servicios personales mediante contenedores.
- Diseñar una arquitectura NAS para almacenamiento y backups.
- Evolucionar hacia Proxmox como plataforma de virtualización.
